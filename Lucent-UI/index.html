One UI suite that dominates 8 niches at once

Lucent UI is a procedural + dynamic uGUI toolkit built on a single backend-neutral DrawList spine: procedural shapes, soft shadows & GPU liquid-glass, animated buttons, live charts, a zero-GC virtualized scroller, RTL/Arabic bidi text (no TextMeshPro required), a 25+ widget library, and runtime theming & localization — each one strong enough to stand alone.

The interactive demo — one draw call, zero textures, 0 B/frame GC. Rendered on the GPU.

Eight niches, one asset

Every panel below is real, shipped, and individually competitive.

N1

Procedural Shapes

Rounded-rect, circle, star, ring & polygon with per-corner radius, gradient/pattern/sprite fills and crisp analytic AA.

N2 · N3

Effects & Liquid Glass

Drop/inner shadow, glow & outline on any uGUI graphic, plus GPU dual-Kawase blur and frosted glass.

N4

Animated Buttons

Selectable + channel stacks, 35+ presets, ripple, audio and entrance animation — zero setup.

N5

Charts & Graphs

Bar/line/area/pie/scatter/heatmap/candlestick with axes, legends, tooltips, dashboards and export.

N6

Virtualized Scroll

A 0-GC recycler — 5000 items, ~9 live cells — with snapping, parallax and gamepad navigation.

N7

RTL & Animated Text

UAX #9 bidi + Arabic shaping, SDF/MSDF and per-glyph animation — no TextMeshPro required.

N8

Component Library

25+ theme-aware widgets: accordion, dropdown, modal, tabs, toast, date picker, 300+ procedural icons.

+

Theming & Localization

Design tokens, a runtime theme switch that recolors the whole UI, and an I2-parity localization term DB.

Why it’s fast

Built for the frame budget, not just the screenshot.

  • One mesh = one draw call (effects batch into the shape)
  • Zero per-frame GC on the virtualized scroller
  • Analytic, resolution-independent anti-aliasing
  • GPU liquid-glass on URP, Built-in & HDRP
  • Imports on any input backend (New Input System optional)
  • Procedural by design — no texture atlases to manage
